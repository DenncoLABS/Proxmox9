# Install

Install, uninstall, and health-check scripts for Proxmox9 custom work.

Use this folder for scripts that safely apply Dennco extensions, overlays, or packages to a test or production node.

Recommended files:

- `install-extension.sh`
- `uninstall-extension.sh`
- `health-check.sh`

Do not put secrets, API keys, node passwords, or production-only values in this repo.
