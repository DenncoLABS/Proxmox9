# Apps

Use this folder for standalone applications that integrate with Proxmox VE 9.

Examples:

- Management dashboards
- Automation tools
- API clients
- Billing or customer portals
- Provisioning systems
- Monitoring utilities

## When to move an app to its own repo

Move an app out of this repository when it becomes a product with its own:

- Release version
- README and documentation
- Installer or package
- CI/CD pipeline
- Customer/user-facing issue tracking

Until then, this folder is fine for prototypes and internal tools.
